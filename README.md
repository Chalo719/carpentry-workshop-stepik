# Carpentry Workshop Лендинг

Одностраничный сайт для **столярной мастерской** — проект, выполненный в рамках обучения на Stepik. Демонстрирует услуги, опыт и работы мастерской.

## О проекте

Лендинг презентует столярную мастерскую: информацию о компании, блог со статьями (строительство мебели, инструменты, опыт) и портфолио работ. Проект отличается адаптивным дизайном и мобильным меню-бургером.

## Технологии

`HTML` `CSS` `JavaScript` `Flexbox` `Адаптивная вёрстка`

## Что реализовано

- Фиксированная шапка с навигацией и плавным скроллом к секциям
- Вступительная секция с заголовком на фоновом изображении
- Блок «О нас» с текстом и фотографией
- Карточки блога (3 статьи) с изображениями
- Секция «Работы» с примером портфолио
- Бургер-меню для мобильных устройств (JS)
- Адаптив под планшеты и телефоны
- Плавная прокрутка по якорям
